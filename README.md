# atui
_This app is only available in Japanese. The following is all written in Japanese._
Atui is the browser assistant made by japanese high school student.

## About Us
Atuiは日本の高校生が一から開発したアシスタントです。
これは現在サポートされている、すべての端末・OSで動くことを目標に作られました。
<sub>動作確認済みOS：Windows10, Windows11, Linux(Ubuntu), Android16~13, iOS 26~15</sub>

## What Can I Do?
AtuiではAIなどの"なんでもできる"プログラムをまったく使用しないため、利用できる機能はAIに比べたら限られています。
しかし、Atuiの一部の機能はAPIからデータを取得して動作していますので軽量かつ必要なデータのみを表示できます。（無駄な情報がない）

以下はAtuiで利用できる機能の一覧です：
`天気`     日本の天気を取得します。都道府県単位はもちろん、地域ごとの天気を調べることができます。livedoor提供の"地域コードに対応しています。"
`電卓`     新規ウィンドウで簡単な計算ができる電卓を表示します。関数計算は現時点では不可能です。
`ＱＲ`     ユーザーが入力したURLまたはテキストをQRコードに変更します。
`勉強`     開発メンバーの一部が別のプロジェクトで過去に作成した勉強をサポートするウェブアプリに誘導します。
` 暇 `     暇つぶし動画。海外では有名なあの動画が開きます。
`しりとり` ボットとしりとりができます。しりとりはローカルで動きますが、まだまだボットが知っている日本語が少ないため、ユーザーが強制的に敗北されるケースが確認されています。ご了承ください。
`郵便番号` 郵便番号から地域に変換します。_（例：1008959 → 東京都千代田区霞が関）_ また、その逆にも対応しております。郵便番号は必ず**ハイフンはなし**で入力してください。
